
# Learning PLATFORM

## Getting started
To get the Node server running locally:

```
git clone https://github.com/IbrahimShamma99/LearnWebCommunity
cd LearnWebCommunity/
npm i
npm start
```
## Docker image 
```
sudo docker build -t learnwebcommunity .
```
